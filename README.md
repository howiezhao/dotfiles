My personal software configuration, including:

- Bash
- GDB
- Git
- iTerm2

- Sublime Text 3
- tmux
- Vim
- Windows Terminal
- Zsh

## Install

### Windows

```PowerShell
.\link.ps1
```

### Linux/macOS

```Shell
./link.sh
```

## Uninstall

### Windows

```PowerShell
.\unlink.ps1
```

### Linux/macOS

```Shell
./unlink.sh
```
